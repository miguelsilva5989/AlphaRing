# Contributors and Project Lineage

AlphaRing is an improvement fork. Its implementation and research were developed across several repositories:

- WinterSquire created the original AlphaRing project and core MCC tooling.
- thejackbitt maintained a profile-focused fork used in the later lineage.
- kirklandsig added Proton compatibility, controller binding work, fixes, and the experimental branch used as this fork's base.
- miguelsilva5989 maintains this fork's UI, safer setup workflow, build/install automation, and experimental dual-monitor renderer.
- Priception contributed controller UI support and interface/crash assistance upstream.
- Assembly/XboxChaos research informed tag-group handling.
- Blender's Bezier implementation was referenced by the inherited curve editor.

Git history remains the authoritative record of individual changes. Links and additional details are available in [README.md](README.md#project-lineage-and-credits).
